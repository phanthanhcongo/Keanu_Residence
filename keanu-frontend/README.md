# Keanu Frontend

This is the front-end application for the Keanu project, built with React, Vite, and Tailwind CSS.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
